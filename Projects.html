PROJECTS

Frogger (2017)
St-Lawrence College Project

A Project in our first year was to create a game that is similar to the popular game Frogger. This project used SDL and C++ and also paint for creating my own images.

The game is similar to frogger where the objectives is to get to the life rings while avoiding the cars and the water by standing on the logs and avoiding the cars.

Brandon's Revenge
St-Lawrence College Group Assignment

Brandon's Revenge is a game created as a project which is sound focused. Simular the Slenderman the objective of the game is you need to collect notes to spell a name. once you collect all the notes you win the game.

In this group project. I worked on the pathfinding for the enemy

Money Simulator
UNREAL CLICKER GAME

Money Simulator is a game that I created in unreal where the objective of the game is to click the buttons to create money and buy more stuff with it.

The game was inspired by Adventure Capitalist.

The game can also be played in VR using the Occulus rift.
